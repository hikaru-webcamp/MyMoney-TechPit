# MyMoney-TechPit
TecPit教材であるJavaScriptで家計簿アプリをつくりました

# 目的
JavaScriptの練習or復習

# 機能
・収支の入力  
・入出金履歴の一覧表示  
・入出金履歴の削除  
・支出のグラフ表示  


# 環境
OS：macOS  
ブラウザ：Chrome  
エディタ：VS Code (Visual Studio Code)  
データベース：IndexedDB  
ライブラリ：Chart.js（グラフ作成）、sakura（クラスレスCSS）
